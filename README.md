# Customer Segmentation using K-Means

This project applies **K-Means Clustering** to segment mall customers based on their **Annual Income** and **Spending Score**.  
It helps businesses understand customer groups and tailor marketing strategies accordingly.

---

---

## ⚙️ Features
- Loads customer dataset (`Mall_Customers.csv`)
- Preprocesses data using **StandardScaler**
- Determines optimal number of clusters using the **Elbow Method**
- Applies **K-Means clustering** to group customers
- Visualizes clusters with **Seaborn scatter plots**
- Saves clustered data to `clustered_customers.csv`
